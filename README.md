Sistema de Análisis Gráfico para Casetas

Sistema generador de gráficos y tablas formato .xlsx y .csv para hojas de cálculo de información referente a casetas de cobro de autopistas El formato del documento debe contener las columnas siguientes para la realización correcta del análisis:

    Autopista: Número que identifica a la autopista
    Origen: Es el nombre de la autopista en la que esta la caseta
    Destino: Es la dirección a la que va la autopista
    Vía: Código para definir el tipo de vía
    Modalidad_Vía: La clase para determinar el modo de pago de la vía
    Fecha: Formato AAAA:/MM/DD
    Hora: Formato de 24 hrs o de 12 hrs
    Evento: Código del evento que puede pasar en la caseta (si hay alguna anormalidad o pago normal)
    Modo_de_Pago: El tipo de pago que se presenta en la caseta (tarjeta, efectivo, TAG, etc.)
    TAG Clase: ID del TAG en caso de aplicar
    Clase: Indica el tipo de vehículo que paso por la caseta
    Importe: Lo que se pago de caseta

Las gráficas que genera son las siguiente:

    Ingreso por clase: Muestra los ingresos obtenidos de cada clase por cada origen de autopista
    Frecuencias por hora: Muestra la cantidad de vehículos por clase que transitaron por cada origen de autopista

Entregables: Graficas en formato JPG y tablas compiladas en formato .xlsx
